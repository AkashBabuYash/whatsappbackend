# WhatsApp Clone Backend

This is the **backend** for the WhatsApp Chating project. It handles real-time messaging, video calls, and socket communication.  

Frontend is hosted here: [WhatsApp Clone Frontend](https://era-techlimited.netlify.app/)

---

## 🚀 Features

- Real-time messaging using **Socket.IO**
- Video/audio call support via **WebRTC**

- Simple room-based architecture
- Easy integration with React frontend

---

## 💻 Tech Stack

- **Node.js** – Runtime environment
- **Express.js** – Backend framework
- **Socket.IO** – Real-time communication
